1- En materiales.h se encuentran las estructuras de datos correspondiente a materiales
2- En edificios.h se encuentran las estructuras de datos correspondiente a edificios
3- En utiles.h se encuentran las declaraciones de las funciones con sus correspondientes pre y post condiciones
4- En colors.h se encuentran las definiciones de las constantes para utilizar colores
5- En utiles.cpp se encuentra la implementacion de las funciones
6- En main.cpp se encuenra el código principal del programa
7- En materiales.txt se encuentra la lista de materiales con las respectivas cantidades
8- En edificios.txt se encuentra la lista de los edificios con los parámetos correspondientes
9- El Makefile incluye las instrucciones de compilación para el compilador

________________________________________________________________________________________________________________
    - Para compilar usar el comando make
    - Para correr: ./main
    - Para ejecutar valgind : make valgrind
